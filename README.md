# E-commerce-Price-Tracker-Automation-System-
An automated system that monitors product prices on Amazon and eBay, providing real-time tracking and instant email alerts when prices drop.
🚀 Overview
This project was built to solve the problem of manual price checking on large e-commerce platforms. It combines a robust Python backend for data extraction with a modern Blazor WebAssembly frontend for a seamless user experience.

✨ Features
Dual Platform Support: Scrapes live data from both Amazon and eBay.

Automated Monitoring: Periodically checks for price fluctuations.  

Instant Notifications: Uses SMTP to send email alerts the moment a price drops below a user-defined threshold.  

Interactive Dashboard: A responsive UI built with Blazor to manage tracked products and view price history.

Web Scraping Logic: Custom-built scrapers that handle dynamic content and extract clean pricing data.

🛠️ Tech Stack
Frontend: Blazor WebAssembly (C# / .NET)

Backend: Python

Scraping: BeautifulSoup / Selenium (or specify if you used Scrapy)

Automation: Python apscheduler or cron jobs

Database: fastApi.
